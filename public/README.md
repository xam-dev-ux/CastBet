# Public Assets

This folder contains static assets for the CastBet application.

## Required Images

Please add the following images for Farcaster Frame integration:

### Icons & Images
- **icon.png** (512x512px) - App icon for Farcaster
- **og-image.png** (1200x630px) - Open Graph image for social sharing
- **splash.png** (1080x1920px) - Splash screen for Farcaster Frame

### Favicon
- **favicon.ico** - Browser favicon

## Design Guidelines

- Use the brand color: `#0ea5e9` (sky blue)
- Keep designs clean and professional
- Ensure text is readable on all backgrounds
- Test icons at different sizes

## Farcaster Frame Configuration

Images are referenced in `.well-known/farcaster.json`:
- `iconUrl`: https://castbet.app/icon.png
- `imageUrl`: https://castbet.app/og-image.png
- `splashImageUrl`: https://castbet.app/splash.png
